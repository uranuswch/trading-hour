# Trading Hour Examples

This directory contains example programs demonstrating how to use the `tradinghour` library.

## Running an Example

```bash
go run ./examples/is_open
go run ./examples/timeline
go run ./examples/new_markets
```

## Examples

| Directory | Description |
|-----------|-------------|
| `is_open/` | Check if markets are open at a given time |
| `timeline/` | Get full day schedule for a market |
| `new_markets/` | Examples for Rates, Metals, and FX markets |
